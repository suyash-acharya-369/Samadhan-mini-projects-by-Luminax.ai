# 🎓 Luminax.ai: AI-Powered Student Community

An **AI-powered community platform** where students don’t just study alone — they **compete, collaborate, and share resources** to grow together.  

## 🚀 Features
- 📚 **Community Hub** → Share notes, PDFs, and study resources  
- 🏆 **Gamification** → Leaderboards, streaks, badges, challenges  
- 🤖 **AI Learning Tools** → Quiz/flashcard generator, AI study buddy, smart recommendations  
- 🌐 **Bilingual Support** → Hindi ↔ English (text + voice)  
- 📊 **Analytics** → Progress dashboards, weak-area detection, community insights  
- 🧘 **Wellbeing** → Daily stress check-ins & motivational nudges

## 🛠️ Tech Stack
- **Frontend** → React + Tailwind  
- **Backend** → Node.js / FastAPI  
- **Database** → MongoDB / Firebase  
- **AI APIs & Models** → OpenAI GPT, Whisper, HuggingFace, IndicTrans  
- **Other Integrations** → Notion API, YouTube API  

---

## ⚡ Implementation Roadmap
- **Phase 1 (Hackathon)** → Core features (community hub, AI quiz generator, study buddy, leaderboard)  
- **Phase 2** → Gamification, peer mentorship, bilingual voice support, wellbeing tracker  
- **Phase 3** → Expansion to multiple exams (JEE, GATE, CAT, NDA) + offline access  

---

## 📈 Evaluation Metrics
- Engagement → Active users, streaks, study hours  
- Collaboration → Shared resources, study room activity  
- Performance → Quiz improvements, weak-area mastery  
- Accessibility → % using Hindi/voice features  
- Scalability → Adoption across multiple competitive exams
